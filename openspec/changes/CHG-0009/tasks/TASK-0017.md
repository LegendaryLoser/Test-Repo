# TASK-0017 — Vendor corpora + scaffold benchmark framework (RED)

## Intent

Vendor two real-world corpora (OpenSpec docs MIT; PROMISE NFR CC BY-SA 3.0),
scaffold benchmark loaders/runner/report with NIE stubs, and write real
test bodies. Tests fail at NIE.

## Research notes (input to the proposal)

Online search surfaced these candidates:

- **OpenSpec** (Fission-AI/OpenSpec, MIT) — direct format match for
  markdown-rule dry-run.
- **PROMISE NFR / tera-PROMISE / PROMISE_exp** (Cleland-Huang 2007,
  CC BY-SA 3.0) — 625-969 labeled FR/NFR requirements; widely cited
  research benchmark.
- **NoRBERT** (BERT-based requirements classification) — extension of
  PROMISE with finer labels.
- **EARS-Based Functional Requirements dataset** (MDPI Systems 2025,
  9529 requirements) — license needs verification.
- **PURE dataset** (7445 samples) — license needs verification.
- **Cucumber/Gherkin .feature files** — per-repo MIT-licensed; aggregation
  friction.

Sources cited in PR description; only OpenSpec docs and PROMISE NFR are
vendored in this CHG (license-clear, size-manageable).
