# TASK-0040 — Persist 35 sub-agent audit transcripts + findings (corpus rescue)

## Intent

A fresh session resuming from this branch flagged that the audit corpus
was not durable. The original audit README (TASK-0035) had declared raw
sub-agent outputs would not be persisted, treating the container-local
`/tmp/` cache as the "authoritative raw record." Container reclaim
would have lost the ~700 raw findings, leaving the next session unable
to confirm Tier-D deprecation, re-cluster under refined QD axes, draft
resolution CHGs with full finding text, or use the corpus as evidence
for the methodology codification ADR. Rescue the corpus from
`/root/.claude/projects/-home-user-Test-Repo/<session>/subagents/`
before the container is reclaimed.

## Definition of done

- [x] Source cache located at
      `/root/.claude/projects/-home-user-Test-Repo/<session>/subagents/`
      (5.7 MB across 35 `.jsonl` files + `.meta.json` siblings).
- [x] 35 transcripts mapped to 35 stream prefixes via `.meta.json`
      description fields (the 36th stream, `ARCH-`, was in-context with
      no separate transcript).
- [x] Raw transcripts copied to
      `openspec/_bmad-output/knowledge/audit/2026-05-17-architecture/raw-transcripts/`
      with stream-named filenames (e.g. `EDGE-agent-abf3f95219d0706c3.jsonl`).
- [x] Per-stream findings extracted into `findings/<PREFIX>-findings.md`,
      each containing the initial prompt to the sub-agent + the final
      assistant message (the audit deliverable for that stream).
- [x] `raw-transcripts/MANIFEST.md` generated with one row per stream:
      wave, prefix, method, model, agent ID, raw path, findings path,
      size.
- [x] Extraction tool `raw-transcripts/persist-corpus.py` checked in
      alongside its outputs (auditable + re-runnable).
- [x] Audit `README.md` corrected: the "raw outputs not persisted"
      caveat struck through and replaced with a pointer to the rescued
      corpus + a lesson-for-future-audits note.
- [x] STATUS.md "Persistent audit artifact" list extended with
      `raw-transcripts/` and `findings/`; new Recent-decisions row for
      the corpus rescue.
- [x] Total repo addition: ~6.5 MB across 74 files.
- [x] Commit trailer `Tests-Status: none`.
