## Risk

- **The 42 clusters from `signal-ledger.md` §5 are themselves judgments** (subagent-aggregated; cluster boundaries best-effort). If the cluster set were redrawn under different boundaries (smaller minimum size, different similarity threshold), the cell-occupancy map shifts and saturation % shifts with it.
- **Long-tail singletons would change the saturation% downward.** ~80 singletons would distribute across maybe 5-10 additional cells, most of which would be at the cluster-count-of-1 fringe. These cells would not be saturated (singletons-by-definition don't have multiple methods Pareto-competing). Strict-K=2 saturation might drop from 93% to ~75-80% if singletons were included. Methodology ADR should specify whether singletons count toward saturation denominator.
- **QD-score depends on per-method σ × κ that are themselves estimates** (per qd-triage §2.2 ±10pp σ uncertainty + sole-source approximation κ). True QD-score with full greedy ablation could shift the per-cell rankings.
- **W7 ΔQD-score of 29% is high BUT** the increase is concentrated in 3 cells (V, Y, II opened by SCENFUT). If a second scenario-forward method confirms SCENFUT's Pareto-front as optimal, the W8 ΔQD-score would drop sharply. The high W7 value is signal of a recently-admitted method, not signal of ongoing discovery.
