# TASK-0038 — Quality-Diversity triage of all 36 audit streams; methodology ADR draft clauses

## Intent

The Wave 4 methodology-variance finding (TASK-0037) made COMPOSITE-V2
Gate 6 structurally unachievable. A research-grounded replacement is
needed that admits new methods when they provide signal (user's
constraint) but still produces a defensible stopping rule. Adopt the
Quality-Diversity framework from published literature (MAP-Elites,
Mixture of Complementary Agents, ambiguity decomposition), apply it to
the existing 36-stream corpus to produce per-method scoring + tier
catalog + convergence diagnostic, and draft the clauses the methodology
codification ADR will need.

## Definition of done

- [x] Research basis documented with 12+ citations:
      MAP-Elites (Mouret & Clune 2015), Stock 2025, Multi-Objective
      MAP-Elites (Pierrot et al. 2022), ambiguity decomposition
      (Krogh & Vedelsby 1994), Mixture-of-Agents (Wang et al. 2024),
      Self-MoA (Wang et al. 2025), Mixture of Complementary Agents
      (NeurIPS 2025), Multi-Agent Debate Adaptive Stability (ICLR 2025),

      Auditing Multi-Agent LLM Reasoning Trees (arXiv 2602.09341), and
      Trade-Off Between Diversity and Accuracy in Ensemble Generation.
- [x] Four behavioural axes defined (Lens × Temporal × Decomposition ×
      Severity), ~55 meaningful cells.
- [x] σ (signal rate) and κ (marginal complementarity) defined
      operationally.
- [x] σ × κ Pareto admission rule per cell defined.
- [x] ACGR (Archive Coverage Growth Rate) defined and computed for

      magnitude but invariant to method admission).
- [x] All 36 streams scored: 16 Tier A, ~10 Tier B, 1 Tier C, 4 Tier D

      `qd-triage.md`).
- [x] Limitations documented (§11): σ/κ are estimates not measurements;
      behavioural axes are post-hoc; meaningful-cell denominator is an
      estimate; Tier-D candidates need implementation-audit
      confirmation.
- [x] STATUS.md updated with QD-framework adoption decision and
      ACGR-based convergence projection.
- [x] Commit trailer `Tests-Status: none`.
