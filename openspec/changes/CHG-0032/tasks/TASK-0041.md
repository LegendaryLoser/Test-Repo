# TASK-0041 — Author CHG-0032 envelope retroactively

## Intent

A fresh session asked whether the corpus persistence had also taken
care of the CHG-0032 envelope. It had not. Six commits (TASK-0035
through TASK-0040) had been carrying trailers `Task: TASK-NNNN` and
`Requirements: REQ-AUDIT-0001` that referenced files which did not
exist anywhere on disk. The CHG-0032 envelope was missing entirely,
and the cited REQ-AUDIT-0001 was a fictitious ID. Author the envelope
retroactively to (a) give the dangling trailers something to
dereference and (b) ensure the fresh session has the canonical scope
declaration, exit criteria, and per-task summary it would otherwise
need to reconstruct from commit history.
